# StoryForge Bot

Discord bot for the StoryForge MCSM Launcher.

## Commands

| Command | Description |
|---|---|
| `!info` | Info about the StoryForge launcher |
| `!download` | Get the download link |
| `!season1` | Info about MCSM Season 1 |
| `!season2` | Info about MCSM Season 2 |
| `!help` | Show all commands |
